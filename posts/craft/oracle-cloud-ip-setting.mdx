# 고정 ip

오라클 클라우드 VCN 구성을 마치고 인스턴스를 생성했다면 다음으로는 고정 ip를 사용하도록 설정해줘야 합니다.

만약 고정 ip를 지정하지 않을 경우 아래와 같은 문제점이 생깁니다.
1. 인스턴스를 재시작 하거나 중단할 경우 ip가 계속 바뀜.
2. DNS에 등록할 경우 ip가 바뀌게 되면서 계속해서 DNS 레코드도 업데이트 해야 함.

ip는 한정된 자원이고, 이를 이용하기 위해서는 비용을 지불해야되는데 다행스럽게도 오라클 클라우드에서는 계정당 하나의 고정 ip를 지원해줍니다.

## 고정 ip 설정

고정 ip를 설정하기 위해 ** 네트워킹 > IP 관리 > 예약된 퍼블릭 IP ** 로 이동합시다.

![고정 ip 설정](/img/craft/oci-ip-setting/ip_setting1.png)

ip주소 이름, 컴파트먼트를 지정하면 고정 ip를 할당받을 수 있습니다.

![고정 ip 설정](/img/craft/oci-ip-setting/ip_setting2.png)

짠

![고정 ip 설정](/img/craft/oci-ip-setting/ip_setting3.png)

고정 ip를 할당받고 나면 해당 ip로 설정해기 위해 설정하려는 인스턴스로 이동해줍니다.

![고정 ip 설정](/img/craft/oci-ip-setting/ip_setting4.png)

설정하려는 인스턴스로 이동했다면 아래로 스크롤을 내려 좌측 메뉴에 연결된 VNIC를 클릭해줍니다.

![고정 ip 설정](/img/craft/oci-ip-setting/ip_setting5.png)

아래 이미지와 같은 화면이 나왔다면 인스턴스명을 클릭하면 VNIC 정보가 나오게 되는데

![고정 ip 설정](/img/craft/oci-ip-setting/ip_setting6.png)

좌측 하단에 Resources Ipv4 > ... > 편집 버튼을 클릭해줍시다.

![고정 ip 설정](/img/craft/oci-ip-setting/ip_setting7.png)

새로 할당받은 고정 ip로 설정하기 위해서는 기존에 할당된 ip를 제거하고 새로운 고정 ip를 할당하는 방식으로 설정해야 합니다.

먼저 공용 ip 없음으로 설정을 바꾸고 저장버튼을 클릭합니다.

![고정 ip 설정](/img/craft/oci-ip-setting/ip_setting8.png)
